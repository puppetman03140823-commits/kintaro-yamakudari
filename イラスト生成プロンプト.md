# 金太郎 山くだり — イラスト生成プロンプト集（ハイブリッド用）

AI画像ツール（ChatGPTのDALL·E / nijijourney / Midjourney など）で4枚だけ作って、
このフォルダ（`kintaro-yamakudari`）に **下のファイル名そのまま** で入れてください。
入れたら自動で差し替わります（無ければ今の手描きのまま動きます）。

## 作るファイル（4枚・すべて背景透過PNG）

| ファイル名 | 中身 | 向き・構図 |
|---|---|---|
| `hero.png`  | クマに乗った金太郎 | **右向き**（進む方向）、全身、足が下 |
| `coin.png`  | 小判（金色のコイン） | 正面 |
| `rock.png`  | 岩 | 横から、地面に置く感じ |
| `stump.png` | 木の切りかぶ | 横から |

> ⚠️ 一番大事なのは **背景透過（transparent PNG）**。
> DALL·E等で透過が出ないときは「白背景で生成 → remove.bg で背景を消す」が一番確実です。
> 4枚は **絵柄をそろえる**ため、必ず下の「共通スタイル文」を先頭につけてください。

## できたら

4枚をこのフォルダに入れて「画像入れたよ」と教えてください。
表示サイズや位置（大きすぎ/小さすぎ等）は僕の方で調整します。
さらに「走ってる動き」を強くしたい場合は、`hero.png` を2コマ
（`hero1.png` / `hero2.png` = 足の位置ちがい）でもらえればパラパラ動かせます。
